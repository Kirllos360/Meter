# P0-001 — Wire TOU Pricing to Billing Engine

## Goal
Connect existing Time-of-Use (TOU) pricing structure in TariffCharge/TariffChargeDetail to the billing engine so invoices calculate period-specific rates.

## Scope
- Read TOU periods (peak/off-peak/shoulder) from TariffChargeDetail
- Match reading timestamp to TOU period
- Calculate consumption per period
- Apply period-specific rates
- Generate separate invoice line items per period
- Fall back to flat rate if no TOU periods configured
- Does NOT include: demand charges, penalties, credit notes, UI changes

## Business Rules
- TOU periods are defined per tariff in TariffChargeDetail (period_start, period_end, rate)
- Reading timestamp determines which period a reading falls into
- When multiple periods exist in one billing cycle, consumption is split proportionally
- Each period appears as a separate invoice line item with charge_group=TOU
- If a tariff has no TOU periods, existing flat-rate calculation is used unchanged
- Backward compatible — existing invoices and tariffs unaffected

## Architecture Rules
- Must not break area isolation
- Must not write to Symbiot
- Must maintain backward compatibility

## Database
- Tables: TariffCharge (period_type field), TariffChargeDetail (period_start time, period_end time, rate)
- No new tables needed
- Add index: TariffCharge(tariff_id, period_type) if missing

## Backend Files
- `billing/tariff-calculation.service.ts` — add `calculateTOU()` method
- `billing/calculation-engine.service.ts` — refactor invoice generation to call TOU when applicable

## Backend Rules
- New method signature: `calculateTOU(consumption: number, readings: Reading[], tariffId: string): InvoiceLine[]`
- Must handle edge cases: single period, no periods, all-day same rate
- Fall back to flat rate gracefully
- Audit all invoice mutations

## Frontend
- No changes — TOU lines appear as additional invoice line items via existing InvoiceDetailPage

## Security
- Input validation on all tariff parameters
- No secrets in code

## Performance
- Cache tariff charge details (they change rarely)
- Index tariff lookups
